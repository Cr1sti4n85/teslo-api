# Teslo API

1. Clonar proyecto

2. Ejecutar comando

```
yarn install
```

3. Renombrar archivo **.env.template** a **.env**

4. Cambiar las variables de entorno

5. Levantar la base de datos

```
docker-compose up -d

```

6. Levantar proyecto en modo desarrollo:

```
yarn start:dev
```

7. Ejecutar Seed para llenar la bd

```
http://localhost:4000/api/seed
```
